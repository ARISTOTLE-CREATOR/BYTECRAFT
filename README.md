# ByteCraft

A modern, responsive React landing page built from a Figma design as part of a Frontend Internship Assignment.

## Overview

ByteCraft is a clean and professional agency-style landing page that showcases:

- Modern UI design
- Responsive layouts
- Mobile-first development practices
- Clean and maintainable code structure

## Features

- Responsive design for Desktop, Tablet, and Mobile
- Reusable React components
- Modern landing page layout
- Services section
- Testimonials section
- Newsletter subscription section
- Professional footer navigation
- Clean CSS architecture

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS3
- HTML5
